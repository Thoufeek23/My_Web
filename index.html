Search Engine

Developed a celebrity search engine in C++ that allows users to efficiently find and manage celebrity data based on name, age, field, and popularity by integrating multiple data structures. A Max Heap enables constant-time retrieval of the most popular celebrity, while a Binary Search Tree (BST) supports quick name-based searches. An array-based list (similar to a Hashtable) stores and displays all celebrities, and a stack maintains a history of previously visited profiles for instant backtracking. The project follows a modular structure with separate .h and .cpp files, compiled and linked using a Makefile for efficient execution. This implementation demonstrates my proficiency in C++ data structures, algorithm design, and performance optimization.
